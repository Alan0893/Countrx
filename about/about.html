Description

Need information about a country? This website has the ability to obtain country specific information and display that information in a simplified view. Simply enter your country in the search bar and it will return information relating specifically to the country. Organizes information into section for better visualization.

API

➤ Utilizes the REST Countries API to obtain country information.
➤ Incorporates the Google Maps Embed API to display country map.
